### Installing

A step by step series of examples that tell you how to get a development env running.

1. Install Virtual Env

```
python3 -m pip install --user virtualenv pipenv
```

2. Initialize Env

```
python3 -m venv env
```
3. Activate Env
```
source env/bin/activate
```
4. Install Requirements

```
pipenv install
```
